#  balloon-springboot-demo

#### 项目介绍
balloon-springboot-demo 是 balloon 系列项目，balloon-core 和 balloon-springboot 项目的整合类子，也是一个快速开发脚手架，包含一个后台和 vue 前端


#### 依赖
- balloon-core 1.0.0
- balloon-springboot 0.1.0
- jdk 1.8+
- maven 3.3+





#### 功能
1. 用户管理
2. 部门管理
3. 角色管理
4. 根据用户角色动态菜单管理
5. 用户登录登出，登录超时，强制下线（包含活跃用户token自动续约和短信登录功能）
